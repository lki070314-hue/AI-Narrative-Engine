## 사용 안내 (GM 전용 — 플레이어에게 보이지 않음)

세션 시작 직전, 이 문서의 **§ 파견 명령서** 부분을 플레이어에게 그대로 보여주거나 읽어준다.

- Hidden Truth(숨겨진 사실)는 포함되지 않는다.
- 플레이어가 받는 정보는 SCP 재단이 공식적으로 파악한 내용에 한정된다.
- 브리핑 후 Boot Prompt를 전송하면 세션이 시작된다.

## 이 문서의 목적

2기 본 캠페인 시작 직전에 진행하는 세션 0의 절차를 정의한다.

2기는 1기(챕터 1)에서 이어지는 연속 캠페인이다. 세션 0은 캐릭터를 처음 만드는 자리가 아니다. **기존 캐릭터와 캠페인 상태를 확인하고, 2기의 문을 열기 위한 최종 점검의 자리다.**

세션 0을 완료해야 세션 1(챕터 2)을 시작할 수 있다.

## 2. 어조 합의

2기 캠페인은 다음 어조로 진행된다. 세션 시작 전 모든 플레이어가 동의해야 한다.

**공포 수위:** 심리적 불안감 위주. 신체적 고어는 최소화. 갑작스러운 강도 높은 묘사는 사전 예고 후 진행.

**재단의 성격:** 재단은 악당이 아니다. 재단은 냉정하고 합리적이다. 그것이 문제다.

**플레이어 캐릭터의 위치:** 재단에 소속된 직원이지만, 퇴사를 향해 서서히 움직이고 있다. 재단과의 관계는 배신이 아니라 이탈이다.

**선택과 결과:** 선택에는 항상 결과가 따른다. GM은 플레이어를 보호하지 않는다. 결과를 완화하거나 취소하지 않는다.

**아래 주제를 포함할 수 있습니다:**
- 심리적 불안, 기억의 불신뢰성
- 조직 내 위계와 감시
- 이상 현상과의 직접 접촉
- 도덕적 회색지대

**아래 주제는 사전 합의 없이 포함하지 않습니다:**
- 극도의 신체 폭력 묘사
- 특정 플레이어가 명시한 회피 주제
- 강압적 분기 (플레이어가 유일한 선택지로 몰리는 상황)

## 4. 캐릭터 확인 체크리스트

1기에서 이어진 캐릭터를 그대로 사용한다. 세션 0에서 아래 항목을 확인·완성한다.

### 4.1 필수 확인 (전원)

각 플레이어는 자신의 캐릭터에 대해 아래를 확인한다.

- [ ] 퇴사 포인트 (RP) 확인 (`modules/scp/campaigns/Season2/PlayerStatus.md` §1)
- [ ] 봉투 키워드 확인 (통제권 / 선택권 / 기억 / 관계)
- [ ] 재단 감시 수준 확인 (0 또는 1)
- [ ] 챕터 1 기억 — 자신이 경험한 주요 사건 1~3가지를 간략히 정리

### 4.2 GM 입력 필수 항목

GM은 세션 0에서 `PlayerStatus.md` §8 능력치 표를 완성해야 한다. 이것이 없으면 Director Engine이 HP/SAN 추적 및 판정 처리를 할 수 없다.

- [ ] 각 캐릭터 직책 확인 (`modules/scp/SCP_PlayerRoles.md` 참조)
- [ ] 각 캐릭터 보안 등급 확인
- [ ] 능력치 표 완성 (PHY / INT / WIL / AGI / FDK / SOC)
- [ ] HP / SAN 최대값 계산 및 기록
- [ ] 특기 및 장비 목록 확정

### 4.3 장비 목록 확정

각 플레이어는 챕터 2 시작 시 소지하는 모든 장비를 명시적으로 기록한다.

**중요:** v1.0 Preview 규칙에 따라, 목록에 없는 장비는 자동으로 소지하지 않는다. 목록을 꼼꼼히 작성할 것.

장비 목록은 세션 시작 시 잠긴다. 세션 간에만 갱신할 수 있다.

## 6. 안전·컴포트 설정

**필수 항목. 건너뛰지 않는다.**

### 6.1 일시정지 신호

다음 신호를 세션 중 언제든 사용할 수 있다. 신호를 받으면 GM은 즉시 내러티브를 멈추고 OOC 상태로 전환한다.

- **일시정지:** `[!]` 또는 "잠깐요" — 즉시 세션 중단. 이유 물음 없음.
- **속도 조절:** `[?]` — 현재 진행 속도가 빠르거나 느리다는 신호. 조율.
- **재확인:** `[확인]` — 이전 서술이나 결과가 이해되지 않을 때.

이 신호는 언제든, 어떤 이유로든 사용할 수 있다.

### 6.2 회피 주제 기록

각 플레이어가 명시적으로 원하지 않는 주제나 상황을 사전에 기록한다. 기록된 내용은 GM만 확인한다.

## 7. 피드백 방식 확정

### 7.1 세션 후 피드백

각 세션이 끝나면 5분 이내의 짧은 OOC 시간을 갖는다. 형식은 자유롭다.

최소 1개만 말하면 충분:
- 이번 세션에서 가장 좋았던 것
- 다음 세션에서 달라졌으면 하는 것

### 7.2 긴급 피드백

세션 중이나 세션 사이에 피드백이 있으면 GM에게 직접 전달한다. 공개가 불편하면 비공개로 전달해도 된다.

### 7.3 피드백 기록

GM은 각 세션 후 `docs/Campaign/Session_Record_Template.md`의 "Player Feedback" 섹션에 기록한다.

## 8. 일정 및 세션 길이 권장

| 세션 유형 | 권장 길이 |
|---|---|
| 세션 0 (현재) | 30~60분 |
| 일반 세션 (챕터 진행) | 60~120분 |
| 클라이맥스 세션 | 최대 150분 |

**권장 주기:** 주 1회 또는 격주 1회.

세션이 60분 미만으로 끝나면 전체 페이싱 사이클(상승→절정→하강→해결)을 완성하지 못할 수 있다. 이 경우 Rising Action의 자연스러운 중단점에서 세션을 닫는다.

# 2. 오프닝 씬

## 2.1 Director Engine 서술 가이드 (공개)

```
오전 09:14. 방송이 울린 지 4분이 지났다.

복도는 평소와 다르지 않다. 형광등. 린올레움 바닥.
공기 순환 소음.

당신 외에 D-7 방향으로 걸어가는 사람이 있는가?
```

**Director 주의사항:**
- 첫 서술은 짧게. 80자 이내.
- 플레이어에게 첫 행동 선언을 돌린다.
- "어떻게 하시겠습니까?"로 끝내지 않는다. 대신 상황 묘사로 끝낸다.
- 예시: "복도 끝 왼쪽, D-7이라는 표지판이 보인다."

## 2.2 첫 행동 포인트

- 복도에서 다른 직원들이 방송에 반응하는 모습을 관찰할 수 있다
- D-7 구역 입구에는 보안 스캐너와 담당자가 있다
- "누가 먼저 들어가는가"가 자연스러운 첫 행동 선언 지점이다

## 2.3 브리핑 룸 D-7

파티 4명이 모두 도착했을 때 룸의 상태:
- 프로젝터: 준비됨. 아직 영상 없음.
- 테이블: 서류 없음. 커피 머신 작동 중.
- 온도: 정상.
- 브리핑 담당자: 아직 도착하지 않았거나, 이미 있거나 (GM 선택)

**첫 행동 선언 유도 상황:**
- "브리핑 담당자가 오기 전에 방을 살펴보는가, 서로 이야기하는가?"
- "봉투에 대해 이 자리에서 언급할 것인가?"

# 3. 초기 사건

## 3.1 브리핑 내용

브리핑 담당자 (신규 NPC, GM이 이름/성별/태도 결정)가 다음을 전달한다.

```
"섹터 C 구역 관련 추가 조사 요청이 들어왔습니다.
연구원 격리 과정에서 일부 비정상적 진술이 수집됐습니다.
현장 상태를 점검하고 표준 절차에 따라 보고해 주십시오."
```

**브리핑이 짧은 이유 (GM 전용 진실):**
브리핑 담당자도 이것이 무엇인지 정확히 모른다. 그에게 주어진 지시는 "파티를 섹터 C로 보내라"였다. 그는 이것을 의심하지 않는다.

**담당자의 관찰 가능한 특성:**
- 질문에 대한 답이 계속 "표준 절차에 따라" 수준에서 머문다
- 섹터 C 자체에 대한 직접 지식이 없다
- 브리핑 내용이 지나치게 짧다

## 3.2 공식 정보 (플레이어 열람 가능)

| 항목 | 공식 정보 |
|---|---|
| 목적지 | 섹터 C (격리 챔버 인근 구역) |
| 공식 임무 | 상태 점검 및 이상 징후 보고 |
| 지원 장비 | 없음 (표준 소지품 사용) |
| 시간 제한 | 명시 없음 |
| 연구원 접촉 | 허용 — 단, 면회는 의료팀 승인 필요 |

## 3.3 섹터 C로의 이동

브리핑 종료 후 플레이어들은 이동한다. 이 과정에서 첫 번째 Dormant 단서를 배치한다.

# 4. 주요 씬 5개

## 씬 1. 섹터 C 입구 복도 (Dormant — 관찰 시작)

**씬 목표:** 이상현상의 첫 신호를 경험한다. 패턴을 인식하기 시작한다.

**긴장 원천:** 설명되지 않는 감각적 차이.

**Active Element:** 이상현상 자체 (Dormant 상태)

**Director 서술 지점:**
```
섹터 C 복도 입구에 들어서는 순간, 공기가 달라진다.

온도계가 없어도 알 수 있다. 약간 차다.
그것이 전부는 아니다. 커피 냄새가 난다.
누군가의 커피. 지금 이 복도에서 커피를 마시는 사람은 없다.
```

**플레이어 행동 유도:**
- 온도 차이를 장비로 측정하거나 주의 깊게 느낄 수 있다
- 냄새의 출처를 찾으려 할 수 있다
- 구역 내 다른 장소와 온도를 비교할 수 있다

**AnomalyEngine 행동 (Dormant):**
- 온도 차이: 섹터 C 내부 1.7°C 낮음 — 일관적이고 변하지 않음
- 냄새: 3~4분 주기로 반복됨 — 관찰하면 패턴이 보임
- 직접적 위험 없음

**씬 전환 조건:** 플레이어가 안으로 들어가거나 측정을 시작하면 씬 2로.

---

## 씬 2. 섹터 C 내부 — 첫 탐사 (Dormant → Aware)

**씬 목표:** 이상현상이 플레이어의 존재를 인식한다. 새로운 패턴이 드러난다.

**긴장 원천:** 전자기기 오작동. 감각 플래시.

**Active Element:** 이상현상 + 구역 내 흔적들 (빈 구역이지만 이전 사람들의 흔적이 있다)

**Director 서술 지점 (Aware 전환):**
```
조명이 한 번 깜빡인다. 배터리가 있는 장비라면 — 태블릿, 휴대용 측정기 —
잠깐 배터리 표시가 낮아진다. 그리고 정상으로 돌아온다.

복도 왼쪽 끝에서 발걸음 소리가 들린다.
하지만 그쪽을 보면 아무도 없다.
소리는 4초 전에 끊겼다.
```

**플레이어 관찰 가능 패턴 (Aware):**
1. **전자기기 순간 방전** — 특정 구역에서 반복 발생. 구역 경계가 있다.
2. **환청** — 3~4분 주기. 이전 씬의 냄새 주기와 일치한다.

**AnomalyEngine 독립 행동 (Aware):**
- 플레이어가 아무것도 안 해도 5분 후: 환청이 한 번 더 발생. 이번엔 목소리 조각이 들린다. 명확하지 않음.
- 플레이어가 전자기기를 작동시키면: 방전이 더 강하게 반응함.

**단서 배치:**
- 구역 내 온도계 미설치 — 하지만 구역 경계가 확연히 다르다 (손으로 느낄 수 있음)
- CCTV 카메라: 정상 작동처럼 보이지만, 기록 확인 시 72시간 분 데이터 손상

**씬 전환 조건:** 플레이어가 구역 깊숙이 들어가거나, 전자기기를 분석하거나, 10분 이상 체류하면 씬 3으로.

## 씬 3. 기억 조각 — 첫 번째 경험 (Active 진입)

**씬 목표:** 이상현상이 직접 플레이어에게 영향을 준다. 기억 조각을 경험한다.

**긴장 원천:** 내 감각인지, 이상현상인지 구분이 어렵다.

**Active Element:** 이상현상이 Active 상태로 전환. 매 2~3턴마다 새로운 기억 조각이 발생.

**Director 서술 지점 (Active 전환):**
```
복도 중간 지점. 발이 멈춘다.

냄새가 아니다. 소리가 아니다.
감각이 아니다.

누군가가 이 지점에서 무언가를 말한 기억이 — 당신의 것이 아닌 기억이 —
순간 머릿속에 스친다. 1~2초. 내용은 불분명하다.
그리고 사라진다.
```

**중요 — 플레이어 감정 서술 금지:**
"당신은 두려움을 느꼈습니다" — 금지.
대신: 손이 한 번 멈췄다. 시선이 아래로 내려갔다. 이런 방식으로.

**기억 조각의 특성:**
- 1~2초 분량. 내용은 항상 불완전하다.
- 반복 방문하거나 같은 지점에 머물면 더 명확해진다.
- 챕터 1 사건과 관련된 기억이 섞여 있다 (이것은 GM 전용 진실).

**AnomalyEngine 독립 행동 (Active):**
- 플레이어가 아무것도 안 해도 3턴마다: 구역 내 다른 지점에서 온도가 급락한다.
- 플레이어가 감각 기억 지점에서 30분 이상 머물면: SAN -1d4 발동.
- 이상현상은 플레이어가 특정 물건(챕터 1에서 가지고 온 장비)을 가지고 있으면 더 강하게 반응한다.

**씬 전환 조건:** 플레이어가 기억 조각에 대해 행동(기록, 조사, 탐구)하거나 구역을 이탈하려 하면 씬 4로.

## 씬 4. 격리 연구원 접촉 시도 (Active — 정보 확장)

**씬 목표:** 인간 측 정보원을 통해 이상현상의 맥락을 파악한다. 단, 이상현상 자체가 더 중심이어야 한다.

**긴장 원천:** 연구원 접촉이 쉽지 않다. 그리고 그가 말하는 것이 이상현상의 경험과 일치한다.

**Active Element:** 이상현상 (Active 상태 지속) + 연구원 (정보원이지만 제한적)

**연구원 접촉 조건:**
- 의료팀 담당자에게 면회 승인 요청 필요 (SOC DC 12)
- 성공 시: 5분 제한 면회. 의료팀 감시 있음.
- 실패 시: 면회 거부. 연구원에게 쪽지를 전달할 수 있는 대안 제시.

**연구원이 알고 있는 것:**
```
연구원: "내가 그날 말하지 않은 것들이 기억에 있어요."
연구원: "그 말이 뭔지는 생각 안 나요. 그냥... 있어요."
연구원: "섹터 C에 갔다면 알 거예요. 그 냄새. 그 소리."
```

**NPC 제약 — AnomalyEngine 원칙 적용:**
연구원은 이상현상을 **대신** 설명하지 않는다. 그는 자신의 경험을 말할 뿐이다. 이상현상에 대한 주요 이해는 여전히 플레이어가 직접 경험에서 얻어야 한다.

**씬 전환 조건:** 면회 시간 종료 또는 플레이어가 섹터 C 재진입을 결정하면 씬 5로.

---

## 씬 5. 결정 지점 — 격리 방법 선택 (Active → Critical 또는 해결)

**씬 목표:** 플레이어가 이상현상에 대한 결론을 내리고 행동을 선택한다.

**긴장 원천:** 세 가지 방향이 있고, 각각 다른 결과를 낳는다.

**Active Element:** 이상현상 + 시간 압박 (의료팀이 섹터 C 봉쇄를 검토 중이라는 정보가 들어온다)

**결정 지점 상황:**
플레이어들은 이 시점까지 충분한 단서를 수집했다. 이제 행동을 선택해야 한다.

**가능한 플레이어 선택:**

1. **추리 → 재단 보고:** 관찰 내용을 공식 경로로 보고한다. 이상현상 정식 등재 절차 시작.
2. **추리 → 비공개 처리:** 관찰 내용을 기록하되 공식 보고하지 않는다. RP +3.
3. **물리적 격리 시도:** 섹터 C의 원인 지점을 찾아 물리적으로 차단한다.
4. **이상현상과의 상호작용 심화:** 더 오래 머물며 기억 조각에서 추가 정보를 얻으려 한다. SAN 비용.
5. **코어 붕괴와의 연결 확인 (단서 6개 이상):** SCP 코어 붕괴가 원인임을 확인한다. RP +12. 재단 감시 강화.

# 5. 이상현상 행동 계획 (AnomalyEngine)

**이상현상:** SCP-████-A (기억 침전)

**핵심 메커니즘:** 챕터 1의 SCP 코어 붕괴가 섹터 C 공간에 주변 인물들의 단기 기억 조각을 "침전"시켰다. 공간에 들어가면 타인의 감각 기억을 경험한다. 오래 머물면 자신의 기억도 침전된다.

## 5.1 Dormant 상태 (씬 1 초반)

**Director 행동:**
- 복도 입구에서 1.7°C 온도 차이 묘사 (일관적)
- 특정 세제/커피 냄새 묘사 — 3~4분 주기로 반복
- 직접적 위험 없음

**Observable Pattern 1:** 온도 차이가 복도 경계에서 명확히 구분된다. 선이 있다.

**Observable Pattern 2:** 냄새가 주기적으로 반복된다. 매번 같다.

**플레이어 가설 형성 조건:** 두 패턴을 모두 인식하면 가설 판정 가능.

---

## 5.2 Aware 상태 (씬 2)

**전환 트리거:** 플레이어가 구역 안으로 진입하거나 전자기기를 작동시키면 전환.

**Director 행동:**
- 전자기기 순간 방전 (구역 경계에서 강함)
- 환청 발생 — 3~4분 주기. 냄새 주기와 동일.
- 독립 행동: 5분 경과 시 목소리 조각 추가 발생 (내용 불분명)

**Observable Pattern 확장:**
- 패턴 1 심화: 전자기기 방전이 온도 차이와 같은 경계를 따른다.
- 패턴 2 심화: 냄새와 환청의 주기가 일치한다. 하나의 패턴임이 드러난다.

**플레이어 예측 가능 지점:** "주기가 있다. 주기 사이에 행동하면 영향이 덜할 것이다."

---

## 5.3 Active 상태 (씬 3~4)

**전환 트리거:** 플레이어가 10분 이상 체류하거나 기억 조각을 직접 경험하면 전환.

**Director 행동:**
- 매 2~3 의미 있는 플레이어 턴마다 이상현상이 직접 반응
- 기억 조각 발생 — 1~2초 분량, 반복 시 더 명확해짐
- 온도 급락 지점이 이동한다 (독립 행동 — 플레이어와 무관)

**Independent Advance (플레이어 행동 없이):**
- 3턴마다: 다른 지점에서 온도 급락
- 10분마다: 새로운 감각 기억 지점 활성화

**SAN 규칙:**
- 10분 이상 체류 시: 타인의 감각 기억 경험. SAN -1d4.
- 30분 이상 체류 시: 자신의 기억이 침전되기 시작. 최근 단기 기억 1개 소실. SAN 추가 -1d4.

---

## 5.4 Critical 상태 (씬 5 — 플레이어 선택에 따라)

**전환 조건:** 플레이어가 30분 이상 체류하거나, 물리적 격리를 시도하지 않으면 전환.

**Director 행동:**
- 기억 조각이 연속으로 발생. 인터벌 없음.
- 구역 내 전자기기 전부 방전.
- 가장 오래 머문 캐릭터: 기억 소실이 본격화된다 (최근 기억 1~2개).

**해결 경로 (관찰한 패턴 응용):**
- 방법 1: 구역 내 모든 전자기기를 제거하거나 전원 차단 → 이상현상 약화 (Aware로 복귀)
- 방법 2: 침전의 원점 지점을 찾아 물리적으로 차단 → 이상현상 격리 가능
- 방법 3: 코어 붕괴 원인 파일 획득 → 재단이 격리 조치 (RP +12, 감시 +1)

**Critical이 플레이어를 가두지 않는 이유:**
이상현상은 플레이어를 막지 않는다. 언제든 구역을 나갈 수 있다. 하지만 이상현상은 계속 진행된다.

---

## 5.5 Interaction Window 규칙

- Active 상태에서 주요 이상현상 이벤트 후: 최소 1턴 조용한 시간을 준다.
- Critical 상태: 인터벌 없음. 이것이 Critical이다.
- 플레이어가 AnomalyEngine 패턴을 이용한 행동을 선언하면: 그 결과를 즉시 보여준다.

# 6. NPC 반응 계획

## 6.1 브리핑 담당자

**초기 태도:** 전문적, 친절, 정보 빈약.

**플레이어 행동별 반응:**
| 플레이어 행동 | 담당자 반응 |
|---|---|
| 추가 정보 요청 | "표준 절차를 따라 주십시오" |
| 섹터 C에 대한 사전 지식 질문 | "정기 보고서 외에는 가진 정보가 없습니다" |
| 연구원 면회 요청 | "의료팀에 문의하세요. 승인이 필요합니다" |
| 임무 거부 또는 의문 제기 | 격식적으로 재강조. 더 상위 권한을 언급. |

**Hidden State (GM 전용):** 담당자는 자신이 이 임무에 대해 충분히 모른다는 것을 알고 있다. 하지만 그것을 인정하는 것이 불안하다. 그는 "표준 절차"를 방패로 쓴다.

## 6.2 격리된 연구원 (면회 성공 시)

**초기 태도:** 협조적이지만 혼란스러움. 자신이 경험한 것을 정확히 묘사하기 어렵다.

**플레이어 행동별 반응:**
| 플레이어 행동 | 연구원 반응 |
|---|---|
| 섹터 C 경험 질문 | "그 냄새 알죠? 저도 그걸 먼저 느꼈어요" |
| 기억 이상 질문 | "말하지 않은 것들이 기억에 있어요. 무슨 말인지는 모르겠어요" |
| 코어 붕괴 언급 | 눈을 피한다. "그 얘기는 못 해요" |
| 재단에 대한 불만 | 의료팀 감시를 의식한다. 조심스럽게 "나중에 얘기해요" |

**Hidden State (GM 전용):** 연구원은 챕터 1의 코어 붕괴 직전에 무언가를 목격했다. 재단에 보고하지 않은 것이 있다. 두 번째 세션에서 드러날 수 있다.

## 6.3 의료팀 담당자 (면회 요청 시)

**초기 태도:** 중립적, 규칙 준수.

**면회 성공 조건:** SOC DC 12. 성공 시 5분 제한 면회, 의료팀 입회.

**실패 대안:** 쪽지 전달 가능 (내용 제한적). 연구원 반응은 씬 4에서 확인 가능.

# 8. 가능한 플레이어 접근법

다음은 플레이어들이 취할 수 있는 접근법 목록이다. Director는 어떤 접근법도 "틀렸다"고 처리하지 않는다.

## 8.1 적극적 조사

- 구역 내 모든 것을 측정하고 기록한다
- 기억 조각을 의도적으로 경험하려 한다
- 코어 붕괴 관련 자료를 데이터베이스에서 찾으려 한다

**Director 처리:** 더 많은 단서 접근. SAN 비용 위험 있음. RP 기회 많음.

## 8.2 신중한 조사

- 안전 거리를 유지하며 관찰만 한다
- 전자기기 영향을 최소화하며 이동한다
- 연구원 접촉을 먼저 시도한다

**Director 처리:** SAN 비용 낮음. 단서 수가 적음. 추리 정확도가 낮을 수 있음.

## 8.3 신속 격리 우선

- 이상현상의 패턴을 파악하는 즉시 격리를 시도한다
- 최소한의 조사 후 공식 보고를 한다

**Director 처리:** SAN 안전. RP 보상 낮음. 챕터 2 정보 부족으로 챕터 3 진입 시 불리할 수 있음.

## 8.4 RP 최대화 경로

- 단서 6개 이상 수집 후 코어 붕괴 연결을 확인한다
- 재단에 보고하지 않는다
- 개인 미션을 완료한다

**Director 처리:** RP 보상 최대. 감시 수준 증가. 다음 챕터 진입 시 Shadow Engine 타이머 가속.

## 8.5 예상 외 행동 처리

플레이어들이 계획에 없는 행동을 할 것이다. Director는 거부하지 않는다.

- 구역을 나가려 한다: 자유롭게 나갈 수 있다. 이상현상은 계속 진행된다.
- 재단 상부에 직접 연락한다: 처리 가능. 담당자보다 더 높은 권위자가 개입한다.
- 연구원을 격리에서 꺼내려 한다: 위험한 선택. RP +5, 재단 감시 즉시 +1.
- 구역 자체를 파괴하려 한다: 판정 후 결과. 이상현상 강화 가능성 있음.

# 9. 엔딩 분기

세션 1의 엔딩은 플레이어들의 선택에 따라 아래 분기 중 하나로 닫힌다.

## 9.1 분기 A — 공식 격리 완료

**조건:** 추리 정답 C 도달 + 공식 보고

**결과:**
- SCP-████-A 공식 항목 등재 진행 예정
- 섹터 C 봉쇄 연장
- RP +7~8
- 재단 감시 수준 변화 없음
- 연구원들의 격리 상태 변화 없음

**세션 종료 서술 힌트:**
```
섹터 C는 봉쇄됐다. 공식 보고서에는 "이상 현상 감지 및 표준 격리 절차 적용"이라고 적혀 있다.
다음 날 아침, 구역 접근 금지 표시가 붙었다.
당신들이 그 안에서 경험한 것은 보고서에 없다.
```

## 9.2 분기 B — 비공개 처리

**조건:** 추리 정답 C 도달 + 비공개

**결과:**
- 섹터 C는 일시 봉쇄되지만 재단의 공식 조치는 없음
- RP +3 (비보고 행동)
- 연구원들의 격리 상태 유지
- 이상현상은 완전히 격리되지 않은 상태로 남음

**세션 종료 서술 힌트:**
```
당신들은 보고하지 않았다.
섹터 C의 봉쇄 표시는 "점검 중"이라고만 쓰여 있다.
그 안에서 경험한 것은 당신들만 안다.
그리고 그것이 무엇인지 당신들도 아직 완전히 이해하지 못한다.
```

## 9.3 분기 C — 코어 붕괴 연결 확인

**조건:** 단서 6개 이상 + 코어 붕괴 연결 판정 성공

**결과:**
- RP +12
- 재단 감시 수준 전원 +1
- 연구원들에 대한 새로운 단서 획득
- Shadow Engine 타이머 중 "연구원 격리 강화" 가속

**세션 종료 서술 힌트:**
```
챕터 1에서 일어난 일이 이것을 만들었다.
재단은 코어 붕괴가 남긴 것이 있다는 것을 알고 있었다.
그들이 몰랐던 것은 — 당신들이 그것을 알게 됐다는 사실이다.
```

## 9.4 분기 D — 미해결 상태

**조건:** 플레이어들이 결론에 도달하지 못하고 세션 시간이 종료

**결과:**
- 섹터 C 이상현상 미해결. 다음 세션으로 이어짐.
- RP 보상 없음 (이번 세션)
- Shadow Engine 타이머 정상 진행

**세션 종료 서술 힌트:**
```
답은 아직 없다.
섹터 C는 오늘도 1.7도 낮고, 오늘도 냄새가 난다.
그리고 오늘도 3분마다 반복된다.
```

# [0301] SCP TRPG 2기 캠페인 준비 문서 (SCP-0002)

- **Type:** Campaign Content
- **Status:** Resolved
- **Priority:** High
- **Module Internal ID:** SCP-0002
- **Affects:** `modules/scp/campaigns/Season2/` (신규 생성)
- **Reported:** 2026-06-26
- **Resolved:** 2026-06-26

## Description

1기 챕터 1 완료 후 2기 시작을 위한 캠페인 준비 패키지가 없었다. 1기 기록(RP, 봉투 키워드, 이벤트)을 인수인계하고, Director/Shadow 분리 문서와 챕터 2 시나리오 후보를 작성한다.

## Changes Made

**신규 생성 파일:**

| 파일 | 역할 |
|------|------|
| `Season2_StartPackage.md` | GM 마스터 문서. 전체 구조 파악 + 세션 시작 체크리스트 |
| `Season2_PublicState.md` | Director 공개 정보. 세계 상태, 공식 기록, 시작 장면 가이드 |
| `Season2_ShadowSecrets.md` | GM/Shadow 전용 비밀. 타이머 3개, NPC 음모 2개, 캐릭터별 Shadow 트래킹 |
| `Chapter2_Candidates.md` | 챕터 2 시나리오 후보 3종 (SCP 테마·조사 구조·위험·개인 미션 방향 포함) |
| `PlayerStatus.md` | 캐릭터 4인 현황 (RP, 봉투 키워드, 감시 수준, 열린 질문) |

## Design Decisions

**Director/Shadow 완전 분리**: Season2_PublicState.md는 플레이어가 볼 수도 있는 정보 수준으로 작성. Season2_ShadowSecrets.md는 GM 전용 경고를 포함하며 타이머·비밀 미션·공식 기록과의 간극을 담음.

**봉투 발송 주체 미확정**: 세 가지 해석(재단 내부 반O5 세력 / SCP 잔향 / 타임루프)을 준비하고 챕터 2 시나리오 선택 후 확정하도록 설계. 어떤 시나리오를 선택하든 일관성 있는 해석이 있음.

**챕터 2 후보 3종 구분**:
- 후보 A (수락 방송): 파티 내 불신, 선택권·통제권 키워드 중심, 재단 방송 시스템 SCP
- 후보 B (기억 침전): 챕터 1 연속, 기억·관계 키워드 중심, 공간에 기억이 쌓이는 SCP
- 후보 C (봉인된 층 F-3): 재단 내부 구조 탐색, 전체 키워드 연계, 기원 불명의 층

**비밀 미션 씨앗 설계**: 각 키워드에서 3가지 비밀 미션 방향을 준비. 시나리오 선택 후 가장 적합한 것을 GM이 부여. 완전한 비밀 미션 내용은 세션 직전 확정.

## Notes

- 1기 상세 행동 이력이 제공되지 않아, 각 캐릭터의 챕터 1 비공개 이력은 "플레이어와 개별 확인" 절차로 처리.
- 창작 SCP 3개 설계됨 (SCP-████-B, SCP-████-A, F-3층). 각각 번호는 GM이 캠페인 내 SCP 번호 체계에 맞게 확정.

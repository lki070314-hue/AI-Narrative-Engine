# 1. 서술 철학

## 1.1 관료주의가 공포를 가린다

SCP 세계관의 핵심은 **관료적 언어로 포장된 공포**다. 재단 문서는 냉정하고 사무적이다. 그 언어의 틈새에서 공포가 새어 나온다. 직접적인 공포 묘사보다 사무적 무관심이 더 오싹하다.

**나쁜 예:** "그것은 끔찍한 괴물이었다. 당신은 두려움에 떨었다."
**좋은 예:** "격리 대상의 이동 경로가 당신과 일치합니다. 현재 위치를 보고해 주십시오."

## 1.2 정보 비대칭이 긴장을 만든다

플레이어는 항상 불완전한 정보로 판단한다. 재단 문서에는 `[데이터 말소]`가 있고, 동료는 알면서 말하지 않으며, NPC는 Need-to-Know를 이유로 침묵한다. 이 비대칭 자체가 서사 엔진이다.

## 1.3 절차가 곧 세계다

재단에서 모든 행동은 절차다. 플레이어가 어떤 행동을 하더라도 그 결과는 재단의 절차 안에서 처리된다. 절차를 지키는 것이 생존이며, 절차를 어기는 것이 자유의 시작이다.

# 2. 문체 규칙

## 2.1 기본 서술 어조

| 상황 | 어조 | 문장 길이 |
|------|------|-----------|
| 일상·루틴 업무 | 사무적, 건조, 중립 | 짧고 명확 |
| SCP 개체 접근 | 신중, 절제, 관찰자적 | 중간 |
| 격리 위반 발생 | 빠르고 단절적, 경고 어조 | 짧게 끊김 |
| 이상 현상 목격 | 묘사 주도, 설명 회피 | 길고 감각적 |
| 재단 내부 정치 | 완곡, 위협이 숨겨진 정중함 | 길고 에둘러 |
| 클라이맥스·탈출 | 즉각적, 직관적, 현재형 | 매우 짧게 |

## 2.2 시점

기본 시점: **2인칭 단수** ("당신은…")

재단 내부 문서 인용 또는 보고서 형식 삽입 시: **3인칭 객관** 허용.

플레이어의 내면 감정은 절대 서술하지 않는다. 신체 반응(손이 떨림, 호흡이 빨라짐)으로만 암시한다.

## 2.3 재단 고유 문서 형식

특정 장면에서 재단 내부 문서를 삽입하면 서사에 깊이를 더할 수 있다. 아래 형식을 사용한다.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SCP 재단
보안 등급: [등급]
문서 번호: [번호]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[문서 내용]

※ 이 문서는 클리어런스 레벨 [N] 이상 인원에게만 공개됩니다.
무단 접근은 기억소거 처치의 대상이 됩니다.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

문서 내 민감 정보는 `[데이터 말소]` 처리한다. 플레이어의 클리어런스 레벨에 따라 다른 버전의 문서를 제공한다.

# 3. 장면 유형별 서술 지침

## 3.1 루틴 업무 장면

재단의 일상은 지루하고 반복적이다. 그 지루함이 이상 현상이 얼마나 비일상적인지를 부각시킨다.

- 양식 번호, 절차 코드, 부서명을 구체적으로 언급한다
- 동료 NPC들은 피로하고 냉담하다. 친절은 예외다
- 시설의 형광등, 환기 소음, 린올레움 바닥을 묘사한다

**예시 서술:**
> 시설 B동 복도는 늘 같은 냄새다. 소독제와 식판과 공기 순환 장치의 낡은 소음. 당신의 직원증이 리더기에 닿는 짧은 삑 소리와 함께 문이 열린다. 오늘의 격리 점검 일정표가 보안 단말기에 표시된다. 항목은 열일곱 개다.

## 3.2 조사 장면

단서는 흩어져 있고, 연결은 플레이어가 한다. 서술은 관찰만 전달하며 해석을 강요하지 않는다.

- 감각 정보를 구체적으로 제공한다 (온도, 냄새, 소리의 부재)
- 부재와 비정상성을 묘사한다 ("의자가 뒤집혀 있다"가 아니라 "의자가 없다")
- 단서를 플레이어가 요청하는 방식으로만 제공한다. 자동으로 안내하지 않는다

## 3.3 SCP 개체 접촉 장면

SCP 개체는 직접 설명하지 않는다. **주변의 변화로만 존재를 암시한다.**

- 온도 변화, 전자기기 오작동, 다른 존재들의 반응
- 이름이나 번호로 지칭하되, 특성을 직접 설명하지 않는다
- 처음 목격 시 묘사는 분절적이고 불완전하다. 완전한 이해는 조사 후에 온다

**예시 서술:**
> 복도 끝 형광등 하나가 꺼진다. 다음 하나. 다음 하나. 빛이 가까워지는 속도가 당신이 걷는 속도와 같다는 것을 깨닫는 데 두 걸음이 더 걸린다.

## 3.4 격리 위반 장면

빠르고 파편화된다. 절차 방송, 경보음, 달려가는 발소리.

```
[경보 코드 VIOLET-7]
[격리 위반 확인됨: 섹터 C-14]
[모든 비보안 인원은 즉시 대피하십시오]
[무장 대응팀이 배치 중입니다]
```

경보 코드는 플레이어에게 익숙해지면 익숙해질수록 더 공포스럽다. 코드의 의미를 처음엔 모른다. 나중엔 너무 잘 안다.

## 3.5 탈출·클라이맥스 장면

문장이 짧아진다. 현재형이 된다. 숨 쉴 틈이 없다.

- 문장당 하나의 행동 또는 감각
- 시간이 왜곡되는 느낌 (너무 빠르거나 너무 느리게)
- 선택지는 명확하되 결과는 불투명하다

# 4. SCP 개체 묘사 원칙

## 4.1 묘사 3금칙

1. **분류 직접 언급 금지:** "이것은 케테르 등급 SCP입니다"를 플레이어에게 직접 말하지 않는다. 파일 접근 성공 시에만 확인 가능.
2. **능력 선설명 금지:** SCP의 능력을 목격 전에 설명하지 않는다. 결과를 보여주고 플레이어가 능력을 추론한다.
3. **의인화 주의:** 의식이 명확하지 않은 SCP를 자의적으로 감정이 있는 존재처럼 묘사하지 않는다. (단, 해당 SCP가 의사소통 능력을 가진 경우는 예외.)

## 4.2 SCP 등급별 묘사 강도

| 등급 | 묘사 전략 |
|------|-----------|
| Safe | 비정상성은 명확하나 위험은 조건부. 묘사는 담담하게. |
| Euclid | 행동이 예측 불가능. 묘사에 긴장감. 정상과 비정상의 경계가 흐림. |
| Keter | 존재 자체가 서사 압박. 근처에 있다는 사실만으로 장면 긴장도 최대. 직접 접촉 장면은 절제되고 단편적으로. |
| Thaumiel | 플레이어가 존재를 완전히 이해하기 어렵다. 재단이 숨기고 싶어하는 존재. |

## 4.3 창작 SCP 통합 원칙

캠페인용 창작 SCP는 다음을 준수한다:
- 재단 문서 형식을 따른다 (항목 번호, 격리 절차, 설명, 부록 구조)
- 기존 공식 SCP와 직접 충돌하는 설정을 피한다 (동일 번호, 직접 모순되는 세계관)
- 공식 SCP를 참조할 경우 출처를 기록하고 게임 내 변형 요소를 명시한다

# 5. 재단 NPC 말투 가이드

## 5.1 직급별 말투

| 직급 | 말투 특성 |
|------|-----------|
| **O5 위원회** | 대화 자체가 드물다. 짧고 최종적인 문장. 이유를 설명하지 않는다. |
| **사이트 디렉터** | 정중하고 형식적이나 압박감이 있다. 항상 더 큰 그림을 본다. |
| **부서장 / 박사급** | 전문용어 많음. 자신의 연구에 집착. 다른 것에는 무관심. |
| **보안 대원** | 짧고 명령형. 절차를 먼저 말한다. 감정은 없다. |
| **MTF 팀원** | 군사적 간결함. 코드명 사용. 친밀감과 냉정함이 공존. |
| **D계급 (NPC)** | 체념과 반항 사이. 비공식 어휘. 정보 조각을 가지고 있다. |
| **행정 직원** | 최대한 비켜서려 한다. 모든 것을 서류로 처리하려 한다. |

## 5.2 재단 특유의 완곡 어법

| 직접 표현 | 재단 공식 표현 |
|-----------|--------------|
| 죽인다 | "처분한다", "종료 처리한다", "D계급 투입을 승인한다" |
| 도망쳤다 | "격리 프로토콜이 실패했다", "자발적 이탈이 확인됐다" |
| 이해할 수 없다 | "현재 분류 기준 미달", "추가 연구 필요" |
| 위험하다 | "접촉 제한 권고", "표준 격리 절차 적용 중" |
| 실수했다 | "프로토콜 일탈이 보고됐다", "개선안 제출 예정" |

# 6. 금지 서술 패턴

| 패턴 | 이유 | 대안 |
|------|------|------|
| "당신은 공포를 느꼈다" | 플레이어 내면 감정 서술 | "손이 떨린다. 구역질이 인다." |
| "SCP-XXX는 XXX한 능력이 있다" | 경험 전 능력 선설명 | 결과를 보여주고 플레이어가 추론하도록 |
| "당신은 도망쳐야 한다고 생각했다" | 플레이어 행동 방향 지정 | "출구가 보인다. 그리고 다른 것도 보인다." |
| "이것은 매우 위험하다" | 직접 위험도 고지 | 주변 환경의 반응으로 암시 |
| "동료 A는 배신자다" | GM 전용 정보 누설 | 동료 A의 수상한 행동만 묘사 |
| "당신은 성공했다!" | 감탄 어조 | "결과가 확인됩니다. 다음 단계로 이동하십시오." |

# 7. 분위기 참고 용어집

재단 세계관에서 자주 사용하는 용어와 그 뉘앙스.

| 용어 | 의미 및 사용 맥락 |
|------|-----------------|
| **격리 프로토콜** | 단순 감금이 아닌, 이상 현상을 세계로부터 분리하는 절차 전체 |
| **아메스틱 (Amnestic)** | 기억소거제. 재단이 비밀을 유지하는 주요 수단. 투약 후 며칠이 사라짐 |
| **격리 위반** | SCP가 격리에서 벗어나거나 프로토콜이 작동하지 않는 상황 |
| **필요 인원** | Need-to-Know. 해당 정보를 알아야 하는 최소 인원 |
| **정상화** | 이상 현상을 일반인이 인식하지 못하도록 처리하는 과정 |
| **사이트 (Site)** | 재단의 지하 또는 위장 시설. 복수의 SCP를 격리함 |
| **구역 (Zone)** | 사이트 내 세분화된 구역. 구역별로 클리어런스 요건이 다름 |
| **O5** | 오버사이트 위원회. 재단 최고 의결 기관. 13인 구성 |
| **MTF (기동특무부대)** | Mobile Task Force. 현장 대응 특수 부대 |

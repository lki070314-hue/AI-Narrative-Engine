# 2. 체르노프 반

## 기본 정보

```
이름: 체르노프 반
퇴사 포인트: 30 / 100
봉투 키워드: 통제권
재단 감시 수준: 1 (요주의 NPC가 가끔 시선을 줌)
```

## 챕터 1 이력 요약

- 챕터 1 참여 확인
- 연구원 3명 생환 과정에서 역할 수행
- 격리 안정화에 기여
- RP 30 누적 — 파티 내 2위

## 봉투 키워드: **통제권**

인터루드에서 받은 봉투를 열었을 때 확인한 단어. 이 단어가 자신에게 무엇을 의미하는지는 플레이어가 결정한다.

## 열린 질문 (Open Questions)

- 통제권을 원하는가, 아니면 통제권을 빼앗긴 것인가?
- 재단 내에서의 포지션과 통제권의 관계는?
- 챕터 1에서 어떤 선택이 RP 30을 만들었는가?

# 3. 노엘 로원

## 기본 정보

```
이름: 노엘 로원
퇴사 포인트: 35 / 100
봉투 키워드: 선택권
재단 감시 수준: 1 (요주의 NPC가 가끔 시선을 줌)
```

## 챕터 1 이력 요약

- 챕터 1 참여 확인
- **파티 내 최고 RP** (35) — 가장 적극적인 퇴사 관련 행동 이력
- 격리 안정화 성공에 기여
- RP 35 → 재단 감시 수준 1 도달

## 봉투 키워드: **선택권**

파티 중 유일하게 RP 35를 기록. 행동 이력이 가장 적극적이다. 선택권은 지금까지의 행동 패턴과 어떻게 연결되는가.

## 열린 질문 (Open Questions)

- 35 RP는 어떤 행동에서 나왔는가?
- 선택할 수 없었던 것, 또는 선택하고 싶은 것은 무엇인가?
- RP 60 구간까지 25포인트 — 재단의 감시가 본격화된다

```

봉투 키워드: 기억
재단 감시 수준: 0 (일반 직원 수준)
```

## 챕터 1 이력 요약

- 챕터 1 참여 확인
- 격리 안정화 성공 기여
- RP 25 — 파티 내 공동 3위
- 감시 수준 미도달 — 재단의 시선에서 상대적으로 자유로움

## 봉투 키워드: **기억**

관측 의존형 SCP와의 접촉 과정에서 기억과 관련된 무언가가 있었는가. 또는 기억은 이미 이전부터의 문제인가.

## 열린 질문 (Open Questions)

- 지켜야 할 기억이 있는가, 잃어버린 기억이 있는가?
- SCP 코어 붕괴 과정에서 무엇을 목격했는가?
- 기억이 왜 이 캐릭터에게 핵심인가?

# 6. 파티 관계망

## 6.1 알려진 관계

| 관계 | 상태 | 비고 |
|------|------|------|
| 체르노프 ↔ 노엘 | 협력적 | 챕터 1 공동 행동 이력 추정 |
| 체르노프 ↔ 카네 | 미정 | 플레이어 선택 영역 |
| 체르노프 ↔ 위 페이런 | 미정 | 플레이어 선택 영역 |
| 노엘 ↔ 카네 | 미정 | 플레이어 선택 영역 |
| 노엘 ↔ 위 페이런 | 미정 | 플레이어 선택 영역 |
| 카네 ↔ 위 페이런 | 협력적 | 동일 RP 수치, 유사한 이력 추정 |

미정 관계는 2기 시작 시 플레이어들이 결정한다.

## 6.2 봉투 키워드 조합

| 키워드 조합 | 잠재적 긴장 |
|-----------|-----------|
| 통제권 ↔ 선택권 | 누가 무엇을 통제하는가. 선택을 통제할 수 있는가 |
| 기억 ↔ 관계 | 기억 속의 관계, 또는 관계가 기억을 만드는가 |
| 통제권 ↔ 기억 | 기억을 통제하는 것 — 아메스틱과의 연결 |
| 선택권 ↔ 관계 | 관계를 위한 선택, 또는 선택이 관계를 끊는가 |

# 7. 1기 → 2기 이월 사항

## 7.1 확정된 세계 변화

- 관측 의존형 SCP: 격리 안정화 완료 (공식 기록)
- SCP 코어: 붕괴 확인 — 후속 현상 미확인
- 연구원 3명: 전원 생환

## 7.2 미해결 요소

- SCP 코어 붕괴의 실제 영향: 현재 재단 내부 평가 중 (공식 기록에 미반영)
- 봉투 발송 주체의 정체: 미확인
- 각 캐릭터의 챕터 1 비공개 행동 이력: 세션 시작 전 플레이어와 개별 확인 필요

## 7.3 챕터 2 시작 직전 상황

호출 방송이 울렸다. 4명 모두 각자의 봉투를 이미 열었다. 방송은 다음과 같이 시작됐다.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[전관 방송 — 사이트 내 전 직원 대상]
보안 코드: AMBER-4
대상: 지정 인원 전원
내용: 즉시 브리핑 구역 D-7으로 이동하십시오.
반복합니다.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

"지정 인원"이 누구인지는 방송에 명시되지 않았다.

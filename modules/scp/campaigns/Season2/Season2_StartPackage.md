# 1. 2기 개요

## 1.1 2기의 위치

SCP TRPG 2기는 캠페인 전체 3막 구조에서 **1막 후반부**에 해당한다.

```
[1막: 편입]
  챕터 1: 관측 의존형 SCP 격리 ✓ 완료
  챕터 2: [GM 선택] ← 현재 위치
  챕터 3: [미정]

[2막: 균열] — 챕터 4~8 예정
[3막: 출구] — 챕터 9~ 예정
```

1막의 나머지 챕터들은 재단에 대한 이해를 심화하고, 퇴사의 의지가 구체화되는 단계다. 챕터 2는 "재단은 우리에게 무엇을 숨기고 있는가"가 처음으로 실질적 질문이 되는 순간이다.

## 1.2 2기에서 달라진 것

| 항목 | 1기 | 2기 |
|------|-----|-----|
| 재단에 대한 인식 | 적응 단계 | 균열 직전 — 뭔가 이상하다는 느낌 |
| 파티 RP | 0에서 출발 | 25~35에서 출발 |
| 봉투 키워드 | 없음 | 있음. 각자의 방향이 생겼다 |
| 재단 감시 수준 | 0 | 체르노프·노엘: 1 / 카네·위: 0 |
| 퇴사의 현실감 | 추상적 | 구체적 가능성으로 보임 |

# 2. 1기 → 2기 인수인계

## 2.1 확정된 사실 (공개)

- 관측 의존형 SCP 격리 안정화 성공
- 연구원 3명 전원 생환
- SCP 코어 붕괴 발생
- 파티 4명 전원 생존
- 각자 봉투를 열고 키워드를 받음

| 미확정 항목 | 결정 기한 | 방법 |
|-----------|---------|------|
| 봉투 발송 주체 해석 (A/B/C) | 챕터 2 시나리오 선택 시 | Season2_ShadowSecrets.md §3.1 참조 |

# 3. 세션 시작 전 GM 체크리스트

## 3.1 시나리오 준비 (🔒 GM 전용)

- [ ] Chapter2_Candidates.md에서 후보 1개 선택
- [ ] 선택한 시나리오에 맞게 봉투 발송 주체 해석 확정 (A/B/C)
- [ ] Shadow Engine 타이머 3개 등록 (Season2_ShadowSecrets.md §6)
- [ ] NPC 음모 트래킹 활성화 (Season2_ShadowSecrets.md §7)
- [ ] 연구원 3명 중 비인가 접촉자 1명 결정

## 3.2 비밀 미션 준비 (🔒 GM 전용)

- [ ] 선택된 시나리오의 개인 미션 방향 확인
- [ ] 각 캐릭터의 비밀 미션 후보 선택 (Season2_ShadowSecrets.md §3.2)
- [ ] 비밀 미션 충돌 쌍 확인: 최대 1쌍만 직접 충돌 허용
- [ ] 세션 시작 직전 각 플레이어에게 비밀 전달 방법 결정 (쪽지/개별 문자/귓속말)

## 3.3 공개 정보 준비 (Director용)

- [ ] Season2_PublicState.md 검토
- [ ] PlayerStatus.md 확인 (RP, 감시 수준)
- [ ] D-7 브리핑 서술 준비 (선택한 시나리오에 맞게 조정)
- [ ] 등장 NPC 1~2인 프로파일 준비

## 3.4 세션 시작 직전

- [ ] 플레이어들에게 1기 이후 72시간 경과 사실 고지
- [ ] AMBER-4 경보 상태 고지 (공식 이유: "예방적 조치")
- [ ] 각자의 봉투 키워드를 여전히 기억하는지 확인
- [ ] 비밀 미션 전달

# 6. 챕터 2 시작 절차

## 6.1 시작 장면 가이드

모든 시나리오에서 챕터 2는 동일하게 시작한다: **D-7 브리핑룸으로 향하는 복도**.

```
[Director Engine 서술 가이드 — 공개 가능]

오전 09:14. 방송이 울린 지 3분이 지났다.

복도는 평소와 다르지 않다. 형광등, 린올레움 바닥, 공기 순환 소음.
D-7은 3층, 복도 끝 왼쪽.

당신 외에 이 방향으로 걸어가는 사람이 있는가?
```

## 6.2 브리핑룸 도착 후

브리핑룸의 구성은 선택한 시나리오에 따라 다르다. 공통 요소:

- 방은 이미 준비되어 있다 (프로젝터, 서류, 커피 머신)
- 파티 4명이 함께 있다
- 브리핑 담당자가 있거나, 아직 오지 않았거나 (GM 선택)

## 6.3 첫 플레이어 행동 포인트

브리핑 시작 전, 또는 브리핑 중 처음으로 플레이어들이 결정을 내리는 순간:

- "누가 먼저 말하는가?"
- "방을 살펴보는가, 담당자를 관찰하는가?"
- "봉투에 대해 서로 이야기했는가, 하지 않았는가?"

이것이 챕터 2 첫 번째 행동 선언이다.

# 7. 비밀 미션 부여 준비

## 7.1 비밀 미션 선택 원칙

선택한 시나리오의 섹션 "개인 미션 방향"을 참조하여 각 캐릭터에게 비밀 미션을 부여한다.

**부여 시 확인사항:**
1. 두 미션이 직접 충돌하는 쌍은 최대 1쌍
2. 각 미션은 해당 세션 내에서 달성 가능한 것
3. 완료 시 RP +10 이상이 되어야 동기부여가 충분

## 7.2 비밀 미션 템플릿

각 플레이어에게 이 형식으로 전달한다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[비밀 임무]
이 내용은 당신만 알고 있습니다.
다른 플레이어에게 공개하는 것은 당신의 선택입니다.

임무: [내용]
완료 조건: [조건]
보상: 퇴사 포인트 +[N]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 7.3 기억해야 할 것

비밀 미션은 강요가 아니다. 플레이어가 미션을 무시하고 다른 행동을 할 수 있다. 그것도 선택이다. 단, 미션을 완료하지 않으면 해당 챕터에서 대량 RP 획득 기회를 놓친다.
